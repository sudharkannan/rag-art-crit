## What it is
Reference is any external visual source used to inform a drawing — a photo, 
a real object, a posed model, or another artwork. Using reference means 
actively consulting it while drawing rather than drawing from memory or 
imagination alone. Beginners either avoid reference entirely out of a 
misplaced belief that it is cheating, or use it ineffectively by glancing 
at it too infrequently to benefit from it.

## Why reference is not cheating
Every professional artist uses reference. Architecture, fabric, lighting, 
anatomy, perspective — these are complex systems that the brain cannot 
accurately reconstruct from memory alone. Drawing without reference when 
reference is available produces inaccurate results and reinforces incorrect 
mental models. Using reference is how you build the accurate mental library 
that eventually makes drawing from imagination more reliable.

## How to use reference effectively

- **Look more than you draw**: the ratio should be roughly 70% looking 
  at the reference, 30% looking at your drawing. Most beginners do the 
  opposite
- **Understand before copying**: before drawing any element, study it 
  long enough to understand its structure — not just its surface appearance. 
  Ask why it looks the way it does before asking what it looks like
- **Use multiple references**: no single photo captures everything. Use 
  one reference for the overall pose, another for the lighting, another 
  for specific anatomy details
- **Do not trace**: tracing produces accurate outlines but teaches nothing 
  about structure, proportion, or form. Observe and reconstruct instead
- **Reference for imagination**: use reference to study something thoroughly, 
  then put it away and draw from memory. Check your result against the 
  reference and note what your memory got wrong. This is how reference 
  builds imagination

## Where to find reference

- **Pose reference**: Line of Action, SenshiStock, Unsplash, your own photos
- **Anatomy**: Proko's anatomy videos, Anatomy for Sculptors book
- **Lighting**: photograph objects under different light sources yourself
- **Environment and objects**: photograph your own surroundings

## Common mistakes
- Avoiding reference because it feels like cheating
- Glancing at reference once at the start and then drawing from memory
- Copying the surface of reference without understanding the underlying structure
- Using only one reference photo for a complex drawing
- Tracing instead of observing and reconstructing

## Exercise
Choose a subject you regularly draw from imagination — a face, a hand, 
a figure. Draw it once from imagination without any reference. Then find 
three reference photos of the same subject and draw it again consulting 
them actively throughout. Compare the two results and note every specific 
thing the reference version got more accurately. This makes the value of 
reference immediately concrete rather than theoretical.
