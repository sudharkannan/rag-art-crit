# rag-art-crit

An AI-powered (stick with me here) sketch analysis tool that helps aspiring artists improve their work through honest, specific critique and actionable suggestions. NO AI-ART IS GENERATED BY THIS TOOL!
 
---
 
## What 
 
Upload a sketch and get feedback on:
 
- **Technical issues** — proportion problems, shading errors, perspective mistakes, with severity ratings
- **Enhancement suggestions** — ways to make the piece more compelling beyond fixing mistakes (storytelling, lighting, appeal, composition)
- **General tips** — skill-level-aware advice based on what the model observes

DISCLAIMER: Ultimately, this is still a piece of code. It is not human, and may fail to recognize stylistic choices, the emotion of your piece, and similar things that an actual artist could spot instantly.
            Please take critiques and advice with a grain of salt, and ask for help from actual artists when you have the opportunity to!
 
---
 
## Why 
 
I am an engineering student who is attempting to learn digital art as a hobby (@twiiart on twittah baby). I am very bad, but I would like to improve. However, I am too bad to tell exactly what I am bad at. What
is lighting? What is composition? What are values? Is that what's wrong with my drawing, or is it something else? That why I wanted to create a tool that could give specifc advice tailored to YOU and what your art specifically may improve on, as well as guide you in actually improving and getting he most out of focused practice.
 
---

## Why not just generate images
 
AI (and technology as a whole) should be about improving a person's ability to do what they want to do, rather than doing something for them and dulling their skills in the process. AI-generated art is soulless.

## Setup

For non-coders: I'm sorry, but you'll need to get your hands dirty and get a terminal for now :(  You can find tutorials by googling. Once you get that set up you can pretty much just copy paste the code.
    (I'll work on a easier to use web UI when I get time)
 
**1. Clone and create a virtual environment**
 
```bash
git clone https://github.com/sudharkannan/rag-art-crit
cd rag-art-crit
python -m venv venv
source venv/bin/activate
```
 
**2. Install dependencies**
 
```bash
pip install -r requirements.txt
```
 
**3. Set your Anthropic API key**
 
```bash
export ANTHROPIC_API_KEY="sk-ant-...INSERT KEY HERE"
```
 
Add to `.bashrc` or `.zshenv` to persist across sessions.

(You can sign up for a key at console.anthropic.com)

(Cost is about $0.01 - $0.02 per image analysis)

## Usage
 
**Basic analysis:**
 
```bash
python art_analysis.py name_of_sketch.jpg
```
 
**With subject context (recommended for better accuracy):**
 
```bash
python art_analysis.py name_of_sketch.jpg "description of your sketch here"
```
 
Providing a subject description significantly improves accuracy and helps the model not misread intentional costume elements or stylistic choices as mistakes.
 
**Example output:**
 
```json
{

    {
      "category": "proportions",
      "observation": "The torso appears shortened relative to the legs, compressing the figure's midsection",

      "observation": "Lines around the arms are redrawn multiple times, suggesting uncertainty in the stroke",
      "severity": "medium"

  ],

    {
      "category": "appeal",
      "observation": "The pose is symmetrical — pushing one hip out or tilting the shoulders would add dynamism",
      "impact": "high"
    },
    {
      "category": "lighting",
      "observation": "No clear light source is established — committing to one direction would unify the shading",
      "impact": "medium"
    }
  ],
  "general_tips": [
    "Try gesture drawing practice at 30–60 second intervals to build line confidence"
  ]
}
```
